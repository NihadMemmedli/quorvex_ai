# Shot List: Quorvex AI in 10 Minutes: Generate Your First Playwright Test

## Required Product Footage

- Open GitHub repo and docs landing page.
- Show minimal or Docker setup command.
- Open the Quorvex dashboard.
- Create or show a plain-English test spec.
- Run the pipeline.
- Show generated Playwright TypeScript.
- Show validation result and artifacts.

## Capture Notes

- Record at 1920x1080.
- Keep browser zoom between 90% and 110%.
- Use slow cursor movement and pause briefly before clicks.
- Show generated code and run artifacts long enough to read.
- Hide or replace real API keys, credentials, emails, and private project names.
