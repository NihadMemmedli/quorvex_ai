# Tutorials

![Dashboard overview for first-time Quorvex walkthroughs](../assets/ui/dashboard-overview.png)

<p class="caption">Dashboard overview for first-time Quorvex walkthroughs.</p>

Tutorials are learning paths for first-time users. Follow them in order when you want a guided route from setup to a working quality workflow.

| Goal | Start here |
|------|------------|
| Run Quorvex and generate a first UI test | [Getting Started](getting-started.md) |
| Import an OpenAPI contract and generate API checks | [Your First API Test](first-api-test.md) |
| Explore an app and turn discoveries into requirements | [App Exploration & Requirements](first-exploration.md) |
| Learn the dashboard surface | [Dashboard Walkthrough](dashboard-walkthrough.md) |
| Put generated tests into CI | [CI/CD Setup](ci-cd-setup.md) |

## Recommended Path

1. Complete [Getting Started](getting-started.md).
2. Use [Dashboard Walkthrough](dashboard-walkthrough.md) to understand the main screens.
3. Pick either [Your First API Test](first-api-test.md) or [App Exploration & Requirements](first-exploration.md) based on the coverage you need.
4. Finish with [CI/CD Setup](ci-cd-setup.md) once you have generated tests worth running continuously.
