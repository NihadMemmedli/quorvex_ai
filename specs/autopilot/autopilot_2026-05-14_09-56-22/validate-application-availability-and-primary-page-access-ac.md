# Test: Validate Application availability and primary page access accessibility basics

## Description
Checks basic accessible structure for Application availability and primary page access.

## Prerequisites
- Fresh browser session

## Steps
1. Navigate to https://my.gov.az/en/entities
2. Take a browser snapshot
3. Verify visible interactive elements have readable labels or accessible names
4. Verify keyboard focus can move through visible interactive elements

## Expected Outcome
- Visible controls expose accessible labels
- Keyboard focus is not trapped on page load

## Test Data
- Target URL: https://my.gov.az/en/entities

## Source Evidence
- Source requirement(s): REQ-013
